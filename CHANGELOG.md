## [1.0.0] - 2026-03-19

### Added
- 27 deterministic Tauri 2 skills across 5 categories (core, syntax, impl, errors, agents)
- Comprehensive vooronderzoek research document (3700+ lines)
- Definitive masterplan with 10-batch execution plan
- Complete skill catalog (INDEX.md)
- Social preview banner for GitHub
- Validation report with per-skill quality checks

### Categories
- **tauri-core** (3 skills): Architecture, configuration, runtime
- **tauri-syntax** (8 skills): Commands, events, state, window, webview, menu, permissions, plugins API
- **tauri-impl** (10 skills): Project setup, plugin development, multi-window, mobile, build/deploy, testing, migration, security, database, design patterns
- **tauri-errors** (4 skills): IPC errors, build errors, permission errors, runtime errors
- **tauri-agents** (2 skills): Code review, project scaffolding
