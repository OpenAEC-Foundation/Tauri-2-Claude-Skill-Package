# Tauri 2 Claude Skill Package

**Deterministic Claude AI skills for Tauri 2 desktop application development.**

Built on the [Agent Skills](https://agentskills.org) open standard.

## Tech Coverage

| Area | Topics |
|------|--------|
| **Commands** | Tauri commands, IPC, invoke patterns, return types, error handling |
| **Events** | Event system, listeners, emit patterns, payload serialization |
| **Plugins** | Plugin architecture, permissions, capabilities, ACL |
| **Window** | Window management, multi-window, webview configuration |
| **State** | Managed state, AppHandle, async commands |
| **Mobile** | Android/iOS targets, platform-specific code |
| **Build** | tauri.conf.json, bundler, updater, signing |
| **Security** | CSP, allowlist, permissions system, capability files |

## Skill Categories

| Category | Purpose |
|----------|---------|
| `tauri-syntax/` | Command signatures, IPC patterns, event API |
| `tauri-impl/` | Step-by-step development workflows |
| `tauri-errors/` | Error handling, debugging, anti-patterns |
| `tauri-core/` | API overview, version matrix, architecture |
| `tauri-agents/` | Validation, code generation agents |
