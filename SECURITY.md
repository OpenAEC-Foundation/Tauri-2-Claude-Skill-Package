# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability in this skill package, please report it responsibly.

### What to Report

- Skills that recommend insecure patterns.
- Anti-patterns that are missing from error skills.
- Incorrect security advice in any skill.

### How to Report

- Open a GitHub issue with the `security` label.
- Or email the maintainers directly.

### Response Time

We aim to respond within 48 hours and publish a fix within 7 days.

## Tauri Security Model

This skill package teaches Tauri 2's built-in security model including:

- **Content Security Policy (CSP)** configuration
- **Permissions and capability system**
- **Scope-based access control**
- **Isolation patterns**

Skills are designed to promote secure-by-default patterns. See `tauri-impl-security` and `tauri-syntax-permissions` for details.
